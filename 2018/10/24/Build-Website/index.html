将博客部署至 GitHub 上

  • hexo new “Test”: 发表一篇新文章
  • hexo g: 生成静态文件
  • hexo s: 生成本地预览，网址为：http://localhost:4000/
  • hexo d: 部署并提交代码到 GitHub 中
  • hexo clean: 清理缓存文件和已生成的静态文件
  • 最后，访问创建好的 GitHub Page 页面： username.github.io ,此时可以看到自己发布的文章啦。
